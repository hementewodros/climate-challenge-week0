# 🌍 Climate Challenge (Beginner Project)

## 📌 Overview

This project is a beginner-friendly experiment focused on understanding climate change through small, practical challenges. The goal is to explore how everyday actions impact the environment and to learn basic data tracking, analysis, and reflection.

Whether you're new to coding, sustainability, or both—this project is designed to be simple, flexible, and educational.

---

## 🎯 Objectives

* Learn the basics of tracking environmental impact
* Build simple habits that reduce carbon footprint
* Practice beginner-level programming / data logging
* Reflect on personal sustainability choices

---

## 🧪 The Challenge

Pick one or more small climate-related actions and track them over time. Examples include:

* 🚶 Walking or biking instead of driving
* 💡 Reducing electricity usage
* 🚿 Taking shorter showers
* 🥗 Eating fewer animal-based products
* ♻️ Improving recycling habits

Track your progress daily or weekly.

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/hementewodros/climate-challenge-week0.git
cd climate-challenge
```

### 2. Choose your challenge

Decide what habit or metric you want to track.

### 3. Log your data

You can use:

* A simple `.csv` file
* A notebook (e.g., Jupyter)
* A basic script (Python, JavaScript, etc.)

Example:

```
date,activity,value
2026-04-01,bike_km,5
2026-04-01,shower_minutes,8
```

### 4. Analyze (optional)

Create simple graphs or summaries:

* Weekly totals
* Trends over time
* Comparisons
